Close Cockpit

Runs a full quarter-close package from a single upload: an updated deck, a variance analysis derived from the same numbers, and optionally a benchmarking slide and an action item tracker. Everything is reviewable before any file is generated.

1Prior-quarter deck

The PowerPoint to update. Numbers are matched and updated exactly as in Deck Refresh.

2New quarter data

Excel or CSV with the new figures. These same matched values drive the variance analysis automatically.

3Peer benchmark dataOptional

A metric column, a target column, and one or more peer columns. Adds a benchmarking slide to the package.

4Meeting notesOptional

Paste notes and/or upload a text file. Adds an action item tracker, follow-up email, and slide to the package.

Settings

Everything runs locally. No file is transmitted to an external service.
